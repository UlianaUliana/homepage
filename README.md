# Homepage
## Description:
This project is an example of resume. It was created as an educational project for fictitious client - *Donald Trump*.
The aim of the task was to practice in creating simple web-pages using **HTML**, **CSS**, **JavaScript**, lern basic **SEO**, get familiar with **GitHub's Labels**, **Assignees**, and **Milestones** features, master issues and project management.

## Usage 
You can use this project as an example for creating your own resume.
